Mac Address Changer

This tool takes input for a WiFi card and the MAC address you want in the form of arguments and then changes that specified device's MAC address.

Installation

Just clone this repository:

git clone github.com/Abhay-Mahanta/Mac-Address-Changer

Now go to the cloned directory and run python mac_changer.py [arguments]:

cd Mac-Address-Changer

To see how the tool works, run:

python mac_changer.py --help

        Options:
        -h, --help                                     show this help message and exit
        -i INTERFACE, --interface=INTERFACE            Enter interface.
        -m NEW_MAC, --mac=NEW_MAC                      Enter new MAC address.
