Network Scanner

This tool takes input as a range of IP addresses and then scans the network to list available devices.

Installation

Just clone this repository:

git clone github.com/Abhay-Mahanta/Network-Scanner

Now go to the cloned directory and run python network_scanner.py [arguments]:

cd Network-Scanner

To see how the tool works, run:

python network_scanner.py --help

        options:
        -h, --help                        show this help message and exit
        -t TARGET, --target TARGET        Enter the range of IP address.
